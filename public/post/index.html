pandas magrittr style syntax

magrittr introduces some very pleasant syntax to R. It gives you the %>% operator. Whatever is on the left hand side of this operator gets passed into the right hand side. This operator is known as a pipe - similar to pipes in unix. Have a look at this bit of R code, written with the magrittr library. the_data <- read.csv('/path/to/data/file.csv') %>% subset(variable_a > x) %>% transform(variable_c = variable_a/variable_b) %>% head(100) It looks a lot like how you would describe these operations in English.
